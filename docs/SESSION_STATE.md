# GLORIX — SESSION STATE (читать первым, в любом новом чате)

Это единственный файл, который заменяет историю чата. Любой новый чат должен
прочитать его первым и продолжать работу, как если бы чат не менялся —
без повторных вопросов "с чего начать" и без пересказа уже известного.

**Правило обновления**: при каждом значимом изменении (исправлен баг,
принято решение, закрыт пункт аудита) — обновляется секция "ТЕКУЩАЯ ТОЧКА"
ниже, а старая версия этой секции переносится в "ИСТОРИЯ СОСТОЯНИЙ" в самом
низу файла с датой. Это даёт одновременно: (1) актуальный срез сейчас,
(2) полную историю того, как мы сюда пришли — без необходимости читать чаты.

## ИСТОРИЯ СОСТОЯНИЙ (старые срезы, для контекста — не для повторного прохождения)

### 2026-06-19 — Создание этого файла
Контекст: предыдущий чат переполнился (превышен лимит), потребовался перенос
"памяти" в репозиторий, чтобы новый чат не начинал с нуля. До этого момента
вся информация о состоянии проекта существовала только как:
(a) docs/ — архитектура и решения, актуальны структурно, но без среза "что сейчас в работе";
(b) переписка в чате — не сохраняется между сессиями.
Этот файл закрывает разрыв (b) реальным committed-файлом.

Полный аудит (40+ файлов, измеренный размер бандла, протестированные edge-cases
в calcDeposit/calcMarketplaceFee) был проведён непосредственно перед созданием
этого файла. Результаты аудита — это и есть текущий открытый список задач выше.
